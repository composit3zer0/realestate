# Isabel's Place Realty Landing Page

Static landing page for Isabel's Place Realty, focused on South Luzon residential opportunities from Parañaque to Calamba, Laguna.

## Files

- `index.html` is the GitHub Pages homepage.
- `working.html` is a same-file preview you can send or open directly.

## GitHub Pages

In the repository settings, open **Pages**, choose **Deploy from a branch**, select the `main` branch, and use the root folder.

The public URL will look like:

`https://composit3zer0.github.io/realestate/`

## Notes

The contact form is a static frontend preview. Before using the page for real leads, connect it to a form service or backend.
